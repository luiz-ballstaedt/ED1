## Conteúdo do repositório

Este repositório reúne os projetos desenvolvidos na disciplina.

### `sistemaIngressos`

Projeto de sistema de venda de ingressos com uso de estruturas de dados como array, listas encadeadas e fila.

### `trabalho1`

Projeto em C++ para leitura e ordenação de dados de pessoas e datas, com versões usando vetor e lista encadeada.

## Como executar

### `sistemaIngressos`

```bash
cd sistemaIngressos
g++ -std=c++17 -Wall -Wextra -pedantic -o sistema_ingressos sistema.cpp
./sistema_ingressos
```

### `trabalho1`

```bash
make run-array
make run-lista
```
